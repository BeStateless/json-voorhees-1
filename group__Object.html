Detailed Description

The base iterator type for iterating over object types. It is a bidirectional iterator similar to a std::map<std::string, jsonv::value>.

These functions are only applicable if the kind of this value is an object.

Typedef Documentation

typedef basic_object_iterator<object_value_type> jsonv::value::object_iterator

The object_iterator is applicable when get_kind is kind::object. It allows you to use algorithms as if a value was a normal associative container.
