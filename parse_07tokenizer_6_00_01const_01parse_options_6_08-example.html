parse(tokenizer&, const parse_options&)

Reads a JSON value from a buffered tokenizer. This less convenient function is useful when setting parse_options::complete_parse to false.

See also:

  jsonv::parse_options options = jsonv::parse_options().complete_parse(false);
  jsonv::value x = parse(buffered, options);
